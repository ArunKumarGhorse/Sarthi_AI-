# Sarthi AI

Sarthi AI is a basic chatbot project made using HTML, CSS, and JavaScript.  
The chatbot can generate AI responses using API integration and has a simple modern UI.

## How to Run

1. Download or clone the repository

```bash
git clone https://github.com/your-username/Sarthi_AI.git
```

2. Open the project folder

3. Run `index.html` in your browser

## API Key

Add your API key inside `script.js`

```js
const API_KEY = "YOUR_API_KEY";
```
